```
g 移动 z 底部对齐场景
tab 编辑模式
数字 2 边模式
数字 3 面模式 p 独立出来
ctrl b 倒角
ctrl r 环切
i 内插面 e 向内
shift d 复制
/ 独立显示模式
alt w 切割模式 x
alt x 镜像
j 合并
shift s 回到游标原点
```
